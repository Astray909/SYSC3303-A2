Set up instructions:
	1. Extract zip file JiaChenHuang101073186A2.zip
	2. In Eclipse IDE, use Import Projects function to import the directory JiaChenHuang101073186A2
	3. Verify JRE System Library is set to jre-10 or higher
	4. Verify that all source files are placed in src\pbj directory
	5. Run the Server.java file as Java Application first
	6. Verify Server.java is running, run IntHost.java as Java Application
	7. Verify IntHost.java and Server.java are running, run Client.java as Java Application
